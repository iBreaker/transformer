# 👽 Not Only a Translator
本程序不仅仅是一个翻译软件，还没起名字。支持将输入字符转换为另一个字符，包括各种类型的编码解码。将不同类型的转换器封装成插件，方便使用户按需使用。

正在开发中，没有稳定版本...
